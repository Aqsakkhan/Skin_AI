# 🌿 Skin_AI – Smart Skincare Advisor

## 💡 Overview

Skin_AI is an AI-based skincare web application that analyzes user skin conditions and provides personalized, minimal, and budget-friendly skincare routines.

The system focuses on simplicity, safety, and effectiveness by recommending only essential products and optional home remedies while tracking user progress.

---

## 🚀 Features

* 🧠 AI-based skin analysis (manual input for MVP)
* 🧴 Minimal skincare routine (2–4 products only)
* 💰 Budget-based product recommendations
* 🛒 Product links from platforms like Amazon, Nykaa, Flipkart, and Myntra
* 🌿 Safe home remedy suggestions (optional)
* 🔥 Streak-based routine tracking
* 📊 Progress monitoring and improvement tracking
* 🔄 Routine upgrade after 2 weeks

---

## 🧠 Core Concept

* Focus on **primary skin concern first**
* Avoid unnecessary or expensive products
* Combine **products + safe natural remedies**
* Provide **step-by-step skincare progression**

---

## 🛠️ Tech Stack

* Frontend: React.js + Tailwind CSS
* Backend: Node.js + Express
* Database: MongoDB
* AI Logic: Rule-based (initial), ML integration (future)
